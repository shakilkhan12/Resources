HTML Helper

Before using the HTML helper first you need to load the HTML helper in the controller and you can also load from the autoload file simple goto config folder and then click on autoload.php file. 

<?php 
/*
    * autoload helpers
*/ 
$autoload = [
   'helpers' => ['html']
];
 ?>
