Files Upload Library

Files upload library, upload files/images and do the files/images validations

Files Upload Library Usage

just call the file method and pass them array

<?php 
$data = [
'file_name' => 'image',
'allowed_extensions' => 'jpg|png|gif',
//'allowed_extensions' => 'pdf|docx',
'upload_path' => 'images/',
'label' => 'Image'
];
$this->file($data); ?>

Display file upload error, first of all in the controller call run_file() method if this is true that's means you don't have any errro if false that's means you have an error

<?php
if($this->file_run()){
   print_r($this->file_data['file_name']);    
} else {
    //You have an error
    $this->view('user_view'); 
}
?>
