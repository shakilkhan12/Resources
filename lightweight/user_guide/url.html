URL Helpers

Before using the URL helper first you need to load the URL helper in the controller and you can also load from the autoload file simple goto config folder and then click on autoload.php file. 

<?php 
/*
    * autoload helpers
*/ 
$autoload = [
   'helpers' => ['url']
];
 ?>

link_css helper

link_css helper produce the html external link tag

<?php echo link_css("assets/css/style.css");

//produce 
<link rel="stylesheet" href="assets/css/style.css">
?>

link_js helper

link_js helper produce the html script tag

<?php echo link_js("assets/js/app.js");

//produce 
<script src="assets/js/app.js"></script>
?>
