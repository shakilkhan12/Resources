Database

Important

Before using the database first of all you need to create the database connection, so to create the database connection it's very easy Goto config/config.php file and put your database informations in the config.php file


<?php 

$database = [

    /*
        * Add your database informations
    */ 

    'host'      => '',
    'username'  => '',
    'database'  => '',
    'password'  => ''


];

after creating the database connection now you can communicate with database

Query Helper

if you want to insert/update/delete/select data from database then you can use Query helper 


                <?php 

                $this->Query('Your query');
                //insert query
                $this->Query("INSERT INTO table (column1, column2) VALUES (?,?)", [$value1, $value2]);

                //select query
                $this->Query("SELECT * FROM table)");

                //select where query
                $this->Query("SELECT * FROM table WHERE column = ?", [$value]);

                //delete query
                $this->Query("DELETE FROM table WHERE column = ?", [$value]);

                //update query
                $this->Query("UPDATE table SET column = ?, column2 = ? WHERE column = ?", [$value1, $value2, $value3]);

                 ?>

Select Helper

Select helper only select data from the database table, it's takes two paramters the first parameter is the table which is important and the second parameters is optional for example if you want to pass the where condition then pass the where data in the second parameter like ['id' => 5] 


                <?php 

                $this->Select('Table name', 'options');
                //produces 
                SELECT * FROM table_name
                 ?>

Select_Where Helper

Select_Where helper is simply produce "SELECT * FROM table_name WHERE id = 5" query 


                <?php 

                $this->Select_Where('Table name', 'Where data');
                //produces 
                SELECT * FROM table_name WHERE id = 5
                SEELCT * FROM table_name WHERE email = exmaple@gmail.com
                 ?>

Delete Helper

Delete helper produce the delete query for example "DELETE FROM table_name WHERE id = 5"


                <?php 

                $this->Delete('Table name', 'Where Data');
                //produces 
                DELETE FROM table_name WHERE id = 5;
                DELETE FROM table_name WHERE email = example@gmail.com
                 ?>

Update Helper

Update helper produce the update query, Its takes three parameters the first parameter is the table name second parameter is the SET data and third parameter is the where condition below is the query see the example 


               <?php 

                $this->Update('Table name', 'Set data & columns', 'options');
                //produces 
                UPDATE table_name SET name = 'umer', email = 'umer@gmail.com' WHERE id = 5;
                 ?>

Insert Helper

Insert helper produce the insert query it's takes two parameters the first parameter is the table name and the second parameter is the insert data along with columns 


                <?php 

                $this->Insert('Table name', 'Columns & Values');
                //produces 
                INSERT INTO table_name (column1, column2, column3) VALUES ('value1','value2','value3');
                 ?>

Join Helper

Join helper produce the join query it's takes four parameters the first parameter is the first table name, the second parameter is the second table name and third parameter is the matching condition and fourth parameter is optional which is the join name like LEFT JOIN, RIGHT JOIN by default framework will run the INNER JOIN


                <?php 

                $this->Join('First Table', 'Second Table', 'Matching Condition' 'Join Name');
                //produces
                SELECT * FROM First_table INNER JOIN Second_table ON First_table.id = Second_table.id
                SELECT * FROM First_table LEFT JOIN Second_table ON First_table.id = Second_table.id
                SELECT * FROM First_table RIGHT JOIN Second_table ON First_table.id = Second_table.id
                 ?>
