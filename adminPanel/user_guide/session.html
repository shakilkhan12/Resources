Session Library

Session library start the session set the session data, delete the session and destroy the session

Session usage

If you want to set the session data then follow the bellow example, first parameter is the session name and second parameter is the session value

<?php $this->set_session('id','5'); ?>

You can also pass array of session names

<?php $data = ['id' => 5, 'name' => 'Shakil Khan','email' => 'example@gmail.com'];
$this->set_session($data); ?>
Access session data
<?php
echo $this->get_session('id');
 ?>
Set flash message

flash message means for example if user create the account successfully then on the screen normally user dispaly the success message, so bellow is the code how to create flash message

the first parameter is the session name and the second parameter is the flash message

<?php
$this->set_flash('account_success','Your account is created successfully');
 ?>
Access flash message in the view

flash message takes two parameters the first parameter is the session name and second parameter is the CSS class name, CSS class means if you want to style the flash message then you can add the CSS class but it's optional 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
    <style>
        
.alert-success {
    border:1px solid green;
    color: green;
    padding: 10px;
}

    </style>
</head>
<body>
  
<?php $this->flash("account_success", "alert-success") ?>

</body>
</html>
Unset session

unset_session helper/method delete the session, you can unset a spacific session or you can pass array of session names which sessions variables you want to unset

<?php
$this->unset_session('id');
$this->unset_session('name');

//OR

$data = ['id','name'];
$this->unset_session($data);

 ?>
Destroy session

Destroy method delete the entire session

<?php
                    
$this->destroy_session();

 ?>
