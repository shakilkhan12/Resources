Redirect Helper

Before using the Redirect helper first you need to load the Redirect helper in the controller and you can also load from the autoload file simple goto config folder and then click on autoload.php file. 

<?php 
/*
    * autoload helpers
*/ 
$autoload = [
   'helpers' => ['redirect']
];
 ?>

redirect helper

redirect helper produce the php header function

<?php  redirect("controller/method");

//produce 
header(location="controller/method");
